# 🎭 API testing (Python)

This project is a test automation using **FastAPI**, **Python**, and **Pytest**.
It demonstrates end-to-end testing practices, scalable structure, and reporting.

---

## 🚀 Tech Stack

* Python
* FastAPI
* Pytest

---

## ⚙️ Setup & Installation

Clone the repository:

```bash
git https://github.com/AlfredoMc95/API_testing.git
cd TEST_API
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Tests

Run all tests:

```bash
pytest
```

---

## 💡 Key Features

* Scalable and maintainable structure
* Pytest integration
---

## 🧪 Example Test Execution

```bash
pytest test/API/test_items_DELETE.py
```
